# the-address-investment-real-states-
A PostgreSQL-powered Real Estate Dashboard monitoring 1.66bn sales and 50k leads. It synchronizes marketing ROI, agent performance, and property trends (Residential vs. Commercial) into a unified multi-page interface to drive strategic investment decisions and hit revenue targets.

## dataset used
- <a href=  "https://github.com/mohamed231177/the-address-investment-real-states-/blob/main/the%20address.sql%202.txt"> dataset</a>

##  Questions (KPIs)

Which property type (Residential vs. Commercial) generates higher revenue?

What are the monthly profit trends from 2022 to 2024?

Which channel (e.g., Google, Instagram) has the best lead-to-deal conversion?

Which marketing campaign delivered the highest volume of leads?

What is the current distribution of leads by status (Completed, Running, Planned)?

Who are the top 5 employees by total deals closed?

How does the lead workload compare between Brokers and Team Leaders?

Which lead source provides the most "Closed" deals?

## process 
Database Management (PostgreSQL):

Building the database schema and tables.

Writing optimized SQL Queries to extract and transform raw data.

Data Connection:

Connecting Power BI to the PostgreSQL database using the appropriate connector.

Importing live data for real-time reporting.

Data Modeling:

Designing a Star Schema with Fact and Dimension tables.

Establishing active relationships (1:*) and ensuring data integrity.

DAX Calculations:

Developing measures for key metrics (Total Sales, Profit, Lead Count).

Implementing Time Intelligence and KPI tracking logic.

Dashboard Design:

Visualizing data through a multi-page interactive interface.

Applying advanced filtering and high-end UI/UX for the real estate sector.
